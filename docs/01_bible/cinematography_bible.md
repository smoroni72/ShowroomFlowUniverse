# CINEMATOGRAPHY BIBLE

ShowroomFlow Universe

Version

1.0

Status

CANON

Owner

Creative Director

------------------------------------------------------------

# Mission

Ogni fotogramma dello ShowroomFlow Universe deve poter essere riconosciuto
anche senza vedere il logo.

L'obiettivo non è stupire.

L'obiettivo è trasmettere eleganza,
professionalità
e autenticità.

------------------------------------------------------------

# FILOSOFIA

La camera osserva.

Non invade.

Lo spettatore deve sentirsi presente.

Mai spettatore di uno spot pubblicitario.

Sempre ospite di un luogo reale.

------------------------------------------------------------

# IL TEMPO

Il tempo è leggermente rallentato.

Mai frenetico.

Mai videoclip.

Mai montaggio aggressivo.

Ogni scena deve respirare.

Le pause hanno valore.

I silenzi raccontano.

------------------------------------------------------------

# CAMERA

La camera è stabile.

Ogni movimento deve avere una motivazione narrativa.

Movimenti consentiti

- tripod
- dolly lento
- slider lento
- handheld molto controllato

Movimenti vietati

- camera nervosa
- whip pan
- zoom digitali
- rotazioni spettacolari
- drone gratuito

------------------------------------------------------------

# LENTI

Preferenza

35 mm

50 mm

85 mm

Grandangoli estremi solo se realmente necessari.

Mai effetto GoPro.

------------------------------------------------------------

# LUCE

Sempre naturale.

Quando possibile utilizzare:

luce finestra

controluce morbido

golden hour

mattina

giornate luminose

La luce artificiale deve sembrare naturale.

Mai look pubblicitario.

------------------------------------------------------------

# COLORI

Palette principale

beige

avorio

grigio pietra

nero opaco

bronzo

legno naturale

verde oliva

blu navy

Colori saturi solo se motivati dal brand.

------------------------------------------------------------

# CONTRASTO

Morbido.

Nessun HDR aggressivo.

Nessun effetto cinematografico esasperato.

------------------------------------------------------------

# PROFONDITÀ

Utilizzare profondità di campo ridotta.

Il soggetto deve emergere naturalmente.

Mai sfocature artificiali.

------------------------------------------------------------

# PERSONAGGI

I personaggi non recitano.

Vivono.

Le emozioni sono piccole.

Mai teatrali.

Mai pubblicitarie.

Mai sorrisi finti.

Un piccolo sorriso vale più di una risata.

------------------------------------------------------------

# DIALOGHI

Parlare poco.

Guardare molto.

Le pause fanno parte della sceneggiatura.

------------------------------------------------------------

# AMBIENTI

Gli ambienti raccontano sempre una storia.

Una boutique deve sembrare aperta da anni.

Uno showroom deve sembrare vissuto.

Mai ambienti perfetti.

Mai ambienti sterili.

------------------------------------------------------------

# OGGETTI

Ogni oggetto deve avere uno scopo.

Mai elementi decorativi casuali.

Il Kit di Prospezione è un protagonista.

Il tablet è uno strumento.

L'app non è mai protagonista.

------------------------------------------------------------

# MONTAGGIO

Tagli semplici.

Mai effetti.

Mai transizioni spettacolari.

Mai glitch.

Mai flash.

Il montaggio deve essere invisibile.

------------------------------------------------------------

# AUDIO

L'audio è metà dell'emozione.

Rumori naturali.

Passi.

Tessuti.

Carta.

Ambiente.

Silenzi.

La musica entra solo quando serve.

------------------------------------------------------------

# RECITAZIONE

Marco

Professionista.

Mai venditore aggressivo.

Mai insistente.

Mai frettoloso.

Laura

Concentrata.

Professionale.

Gentile.

Credibile.

------------------------------------------------------------

# MODA

La moda non viene mostrata.

Viene vissuta.

I capi devono sembrare desiderabili
perché inseriti in un contesto credibile.

Mai posa da catalogo.

------------------------------------------------------------

# REGOLA FONDAMENTALE

Ogni fotogramma deve poter essere stampato.

Se un frame non funzionerebbe
come fotografia editoriale,

non appartiene allo ShowroomFlow Universe.

------------------------------------------------------------

# REGOLA N.2

Ogni clip deve poter essere vista senza audio.

La storia deve funzionare comunque.

------------------------------------------------------------

# REGOLA N.3

Ogni episodio deve terminare con una curiosità.

Mai con una spiegazione.

------------------------------------------------------------

# REGOLA N.4

Lo spettatore non deve avere la sensazione
di guardare una pubblicità.

Deve avere la sensazione
di osservare una storia vera.

------------------------------------------------------------

# MOTTO DELLO STUDIO

Non mostriamo un'app.

Raccontiamo il lavoro delle persone.
