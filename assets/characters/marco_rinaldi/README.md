# Marco Rinaldi Assets

Marco Rinaldi e' il primo personaggio canonico dello ShowroomFlow Universe.

## Folders

- `turnaround/`: viste principali del corpo o del personaggio.
- `poses/`: azioni, posture e inquadrature ricorrenti.
- `expressions/`: stati emotivi controllati.
- `outfits/`: varianti outfit approvate o candidate.
- `reference/`: sheet, ritratti e riferimenti visuali.
- `deprecated/`: asset non piu' consigliati.

Lo stato canonico vive in `manifest.yaml`.
